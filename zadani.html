Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je skriptovací jazyk, interpretovaný nejčastěji webovými prohlížeči.

JS není roven jazyku Java, oficiálně se nazývá ECMAScript. Existují různé verze, např. ES1 až ES6.

Využití:

  • Používá se k vytváření interaktivních stránek (uživatel může ovlivnit jejich podobu).
  • Umožňuje tvořit dynamické stránky (změna vlastností, animace).
  • Umožňuje vytvořit webové aplikace na straně klienta (browser) i serveru (NodeJS).
  • Lze vytvářet grafické aplikace včetně her.
  • Nabízí mnoho knihoven k nejrůznějšímu použití.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Ending statements with semicolon (;) is not required, but highly recommended.
  2. JavaScript ignores multiple spaces. You can add white space to your script to make it more readable. A good practice is to put spaces around operators ( = + - * / ).
  3. For best readability, programmers often like to avoid code lines longer than 80 characters.
  4. JavaScript statements can be grouped together in code blocks, inside curly brackets {...}. The purpose of code blocks is to define statements to be executed together.
  5. JavaScript keywords are reserved words. Reserved words cannot be used as names for variables.
  6. Single line comments start with //. Multi-line comments start with /* and end with */.

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Vložit kód do samostatného souboru (.js) a připojit v HTML.
<script src="js/main.js"></script>
// 3. Vložit JS jako reakce na událost.
<button onclick="alert('Ahoj')">Klikni</button>

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Vyskočí popup okno (malé okno na vrcholu stránky) obsahující nějaký text, v tomto případě 'Nezadal jsi správné heslo!'. Funguje jako upozornění */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let heslo = prompt('Zadej heslo');

confirm()

/* Vyskočí popup okno které potřebuje potvrzení a zaroveň nám ohlašuje že se v tomto případě chytáme přejí na stránky Seznam.cz */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Nevypadají příliš pěkné, dále nejsou moc oblíbené u uživatelů, proto je většína programů blokuje a taky jsou hodně používány na nekvalitních stránkách.
